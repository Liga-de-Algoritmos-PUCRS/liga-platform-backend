#

## Description

This project is for BEE CRM Client

## Dependecies

- Node.js v22.14.0
- Nest.js 11.0.5
- Docker

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
$ docker compose up -d --build
$ npx prisma generate
```

```bash
# para visualizar seu banco
$ npx prisma studio
```

```bash
# backend
$ npm run dev
```
